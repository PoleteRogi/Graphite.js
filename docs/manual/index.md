# Graphite.js Documentation

Welcome to the Graphite.js documentation.

Graphite.js is a browser-based node editor for building visual logic, data flows, and procedural systems using a canvas-based node graph.

This documentation will guide you through:

- Setting up a node editor
- Understanding core concepts
- Working with nodes and connections
- Creating custom nodes
- Extending the system
